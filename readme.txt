DOS TEXT:

1. Немного о программе:
Теоретически, она должна работать на любой платформе (DOS, i80x86, RAM 640kb).
Надеюсь, что это так.

2. Основные функции:

Новый лист
	создаёт в текущем файле новый лист.
	Ещё существуют функции удаления и печати одиночных листов.

Открыть файл
	открывает или создаёт файл с выбранным именем. Этот файл становится текущим.

Закрыть файл
	закрывает текущий файл (если он был открыт). Текущим файлом становится файл с именем
	ггггммдд.lst (гггг-год, мм-месяц, дд-день), например 20000514.lst. Если он уже был создан,
	то он не стирается.

Печать
	Открывается диалог печати: печать/удаление группы листов в текущем файле,
	до 14 одновременно. Прокрутка осуществляется клавишами вверх/вниз/PageUp/PageDown
	или кнопками "<<"/">>". Выбор производится нажатием чекбокса, стоящего напротив названия.

Добавить файл
	Добавляет к текущему файлу выбранный: теперь из текущего файла доступны листы из обоих.

Редакторы баз данных
	Все работают одинаково. Но: если удалить файл с базой, то редактор создаст новый, и первую
	запись будет неудобно делать. Кнопка "Сохранить" сохраняет текущую статью на её старое место.
	Кнопка "Новый" сохраняет текущую статью в конец файла. ВНИМАНИЕ: сначала напишите статью,
	а уже потом жмите "Новый".
	Диалог "Машины": строка "водитель" теперь работает. Введите первые буквы имени и нажмите
	"Найти". Если эти буквы совпадут с какой-нибудь фамилией, то она появится в строке и
	водитель будет привязан к машине. В том случае, когда водитель не привязан, считается, что
	машина связана с первым в списке.

Редактор листов
	Всё очевидно, но добавлены новые функции: 
	1) при выборе машины выбирается и привязанный водитель (но на всякий случай его лучше
	ещё раз щёлкнуть)
	2) теперь можно ввести в строки "Водитель","Гос.Номер" и "Маршрут" первые буквы названия:
	буде произведён поиск. Если в строке ничего нет, поиск не производиться.

Конфигурация
	можно задать имена файлов баз, имя устройства печати (для принтера - PRN, LPT1 или LPT2),
	каталог для сохранения листов и имя временного файла, используемого функциями удаления.
	Название организации изменить невозможно.
